# n8n Telegram Image AI Bot
Dự án tự động hóa xử lý ảnh qua Telegram sử dụng n8n.

### Tính năng:
- Xóa nền ảnh qua Remove.bg API.
- Chỉnh sửa ảnh qua Stability AI API.
- Tự động đặt tên và lưu trữ lên Google Drive.

### Cách dùng:
1. Import file `.json` vào n8n của bạn.
2. Cấu hình Credentials cho Telegram, Stability AI và Google Drive.
